# GenHerder

A behaviour for avoiding the stampeding-herd problem.

## Installation

The package can be installed
by adding `gen_herder` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:gen_herder, "~> 0.1.0"},
    ...
  ]
end
```

The docs can be found at <https://hexdocs.pm/gen_herder>.
